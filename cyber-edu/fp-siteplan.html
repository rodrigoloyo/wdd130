Site Name

The site name: Human Firewall

Site Purporse

Humnan Firewall website is a educational cybersecurity site for parents to guide and teach kids and youth the responsibility for security in a modern internet and technology accessibility world, also will give tips in general for scenarios that users could avoid get compromised for the lack of security awareness. It will provide links to external websites for articles and statistics about inside the mind of the hacker, and best practices to be safe. Additionally common security gadgets that can be found on Amazon for protecting user devices. It is a free website that might request donations, or subscription monthly magazines and will be sponsored by cybersecurity institutions and sales partners.

Target Audience

The website main target is parents with children in all ages, for teachers and schools that would like to adopt cybersecurity education as part of their program.

Common profile for users entering the website

  • Age: 8 - 105
  • Experience: None
  • Family status: Single, divorce, married

Common scenarios:

  1. Does the website show what is important Cybersecurity? There would be a seasons recommendation sites and link to wheather sites.
  2. Should the user need experience in cybersecurity?This website does no require previous experience, will show basic but important best practices
  3. Should the user have to subscribe to get information?The website is free, however if user wants to enroll for monthly news can susbscribe to the website for a donation cost
